# Week 1 — Exploratory Data Analysis (EDA)
Nesta semana introduzimos o **Exploratory Data Analysis (EDA)**, uma etapa fundamental em ciência de dados para compreender a estrutura, distribuição e relações em um dataset antes de aplicar modelos de Machine Learning.

## Objetivo

Compreender os princípios básicos do **Exploratory Data Analysis** e aplicar técnicas de visualização para identificar **padrões, correlações e outliers em dados reais**.

## Conteúdos

- Introdução ao **EDA**
- Uso de **DataFrames com Pandas**
- Estatísticas descritivas
- Visualização de dados:
  - Histogramas
  - Boxplots
  - Diagramas de dispersão (Scatterplots)
  - Matriz de correlação
