Este repositório contém o material do curso de **Data Science**, incluindo **slides, notebooks e exercícios práticos** ao longo de **8 semanas**.

O curso apresenta os fundamentos de **ciência de dados e machine learning**, combinando teoria, análise de dados e implementação prática em **Python**.

## Objetivo do Curso

Ao final do curso, os estudantes serão capazes de:

- realizar **análise exploratória de dados**
- aplicar **algoritmos de machine learning**
- avaliar e validar modelos
- desenvolver um **projeto completo de ciência de dados**

## Apresentação de Relatórios e Resultados de Pesquisa

Para a comunicação formal dos resultados de investigações ou projetos, os estudantes deverão utilizar o **Template de Paper**, que segue o formato acadêmico padrão:

- Deve conter Resumo, Introdução, Referencial Teórico, Metodologia, Resultados, Discussão e Conclusão.
- Permite apresentar os resultados de maneira estruturada e profissional.
- Deve ser entregue em um único arquivo PDF contendo todas as atividades do laboratório, conforme as instruções do instrutor.

O uso deste template garante uniformidade e facilita a avaliação dos trabalhos.
